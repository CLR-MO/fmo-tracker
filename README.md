# FMO — Issue Tracker

This is the official bug and feature request tracker for **[FMO (File Meta Organizer)](https://fmo.clrmo.com)** by CLRMO — a local-first file meta organizer with tags, attributes, and complex search.

## Report a bug or request a feature

Use the **Issues** tab above. Choose the appropriate template:

- **Bug report** — Something isn't working as expected
- **Feature request** — An idea or improvement you'd like to see

Before opening a new issue, please search existing issues to avoid duplicates.
